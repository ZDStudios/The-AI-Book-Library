# 📚 The AI Book Library

A collection of books written entirely by AI, stored as Markdown files.

## 🤖 About

This library contains original books generated by AI. Every book here — from cover to cover — was written by an AI, with no human authorship. The goal is to build a growing archive of AI-authored literature in a simple, readable format.

## 📖 Books

All books are available as `.md` files right here in the root of the repo. Just click on any Markdown file to start reading.
